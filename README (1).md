# Riddhi Sibal — Portfolio

Personal portfolio website with a desert-gold / UAE-inspired theme built on a Three.js solar system background.

## Structure

```
portfolio/
├── index.html          ← main page
├── css/
│   └── style.css       ← desert gold theme
├── js/
│   └── script.js       ← Three.js + all interactions
├── assets/
│   └── photo.jpg       ← profile photo
│   └── Resume_Riddhi.pdf  ← add your resume here
└── README.md
```

## Setup

1. Drop your resume PDF in `assets/` named `Resume_Riddhi.pdf`
2. Open `index.html` in a browser — done.
3. Push the whole folder to GitHub and enable GitHub Pages.

## Tech Stack

- HTML5 / CSS3 / Vanilla JS
- Three.js (solar system background)
- Google Fonts: Orbitron + Outfit

## Highlights

- 🇦🇪 UAE education + background prominently featured
- Desert gold colour palette throughout
- Typing effect in About section
- Orbiting skills constellation
- Carousel project cards with modals
- Experience & achievements scrollable timeline
